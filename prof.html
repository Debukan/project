Добро пожаловать!

Здесь вы можете увидеть список профессий.

Разработчик ПО: специалист, занимающийся разработкой, тестированием и поддержкой функциональных компьютерных программ. Результатом его деятельности является код, написанный на определенном языке программирования. Для работы необходимо знание алгоритмов, различных языков программирования.

Веб-разработчик (web-developer): специалист в области web программирования, создатель функционирующих онлайн-ресурсов (сайтов, интернет-магазинов, социальных сетей, поисковиков и т.д.). В рамках этой профессии существует несколько специализаций:

developer – отвечает за создание интерфейса, т.е. клиентской части приложения. Он оживляет макеты дизайнеров, делает так, чтобы работали анимации, нажималась кнопки, заполнялись формы с данными. Еще фронтендер публикует продукт в сети и следит, чтобы он правильно смотрелся на разных экранах. Для работы необходимо знание HTML, CSS, JavaScript, требуется погружение в бизнес и требования к продукту;
backend developer – отвечает за разработку серверной части приложений, их внутреннего наполнения. Для работы необходимо знать PHP, SQL, хорошо разбираться в математике;

fullstack developer – совмещает обязанности двух предыдущих специалистов.

Разработчик игр (game developer): специалист, занимающийся созданием программного кода, визуализацией и концепцией игры, а также выбором средств для реализации указанных задач. Для работы необходимо знать различные языки программирования, разбираться в графическом дизайне, аудиоинженерии, обладать креативностью, разбираться в современных играх, а также уметь работать в команде.

Мобильный разработчик (mobile developer): специалист, разрабатывающий программные приложения для различных мобильных устройств (планшетов, смартфонов).

Дата аналитик (data analyst): специалист, занимающийся анализом больших массивов информации в целях подготовки рекомендаций для бизнеса. Для работы необходимо знать Excel, Google таблицы, владеть языком Python, требуются способности к анализу, поиску закономерностей, умение работать с огромными массивами данных.

Специалист по информационной безопасности (cybersecurity): занимается созданием хорошо защищенных систем, препятствует несанкционированным проникновениям и утечке данных. Для работы необходимо знать Linux, владеть языком Python, разбираться в криптографии.

Web-дизайнер (UI/UX): специалист, занимающийся проектированием сайтов и приложений. Для работы необходимы художественные способности, а также знания по психологии.

Специалист по машинному обучению (machine learning engineer): специалист, обучающий роботов (модели машинного обучения) решать конкретные задачи. Для работы требуется хорошее знание математики, владение языком Python.

Тестировщик: специалист, тестирующий приложения и другие программные продукты. Проверяет, как работают разные функции приложений и сайтов в штатных и нештатных ситуациях, описывает ошибки и передает их разработчикам, чтобы исправили.

Системный администратор: специалист, который отвечает за установку, обслуживание и поддержку компьютерных систем. Для работы требуется знать Windows, Linux, macOS, а также разбираться в компьютерном железе.

Специалист по ремонту техники: прекрасно разбирается в компьютерном железе, требуется внимательность.
